•	Licencia para un sitio con un determinado número de computadoras conectadas en red.

	Una vez instalado el software, no podrá ser instalado en otra computadora, pues el funcionamiento se basa en poseer una relación unívoca entre el software y la computadora que lo aloja. •	Puede usar el software en forma comercial.

•	Tiene el derecho de acceder a todas las mejoras que se hagan al software original. Sólo cuando surja una nueva versión que requiere cambios mayores, y por seguridad contra los hackers, se le requerirá al usuario final que se registre nuevamente.
